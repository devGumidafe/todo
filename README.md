# TODO APP

Proyecto TODO realizado en react, material ui y almacenamiento de notas en localstorage.

## Comenzando 🚀

- `npm install` para instalar las dependencias.

```
npm install
```

## Incluye 📦

- Webpack
- Material ui
- Emotion
- Nanoid
- React
- React-DOM
- React-Router v6
